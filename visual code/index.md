## 컴퓨터와 친구가 되자
> 컴공과와 다르게 우리는 컴퓨터에 대해 이해응 시간이 부족하기 때문에 더옥 노력해야한다.

- 하드웨어 (말그대로 단단한 재품 컴퓨터 부품) : 컴퓨터 본체, 모니터, 마우스, 키보드, 등 등 모두 형체가 있는 부품들

- 소프트웨어 ( 하드웨어의 반대) : 우리가 직접 만질수없는 물체의 형태가 없는것, 컴퓨터를 동작하기 위한프로그램 데이트



## 컴퓨터의 중요한 부품
-CPU
-메모리
-하드 디스크


1. `CPU` : 중앙 영산 처리 장치 사람의 뇌와 같이 컴퓨터와 뇌의 역활을 하는 부분이라고 보면 됩니다.
키보드나 마우스 등의 정보를 받아서 처리하는 역활을 해준다.

2. `메모리` : RAM 이라는 단의를 들어본적이 있을건데 작업중인 실행중인 프로크램이나 처리하고 있는 데이트를
나열한 공간

3. `하드 디스크` : 프로크램이나 데이트를 저장하기 의한 저장소 RAM 과 다르게 실핸\ㅇ중인 프로그램이 종료되면 RAM의 해제되지만 하드디스크응 파일로 데이트를 저장해서 컴퓨터가 종료되어도 남아 있는 데이트


### 컴퓨터 프로크래밍이란? 
1. 컴퓨터는 명령한 내용까지만 처리를 한다.
2. 예매한 정보를 x 정확한 정보를 전달해야 한다.
> 컴퓨터에게 명령을 하려면 프로그래밍 해야한다는 말을 들었을 것.
> 컴퓨터는 정해지는 방법을 입력 읽어서 우리의 프로그램을 동작시켜준다.
> 정해진 방법이 없어 소프트웨어를 우리가 생각한 내용으로 구현이 첫 목표 그뒤에 클린코드
> 우리가 이후에 작선할 코드도 절차적으로 처리를 한다.
> 예를 드러 컴퓨터메게 `걸어봐` 라고 명령을 하려면 우리는 경험이 있기때문에 어림짐작해서 걷지만
> 컴퓨터는 전착적으로 알고리즘적으로 정확한 내용을 전달을 하나하나 해줘야 한다.
> 오른발 들어 -> 오른발 내딛어 -> 오른발 땅에 집어 -> 왼발 들어 -> 왼발 내딛어
> 이런식으로 컴퓨터는 ㅂ\전\ㅇ보를 전달 받아서 처리를 하고 세부적인 시나리오를 그릴 수 있는 사람이 되는게 목적
> 개발자적 사고 방식을 기르는 젓이 목표
